ICEfaces ACE Components
Tag ajax

Applied on components that support client behaviors similar to the standard f:ajax behavior.

For more information, see the Ajax Wiki Documentation.

onStartnofalsejava.lang.StringJavascript handler to execute before the ajax request begins. It is passed the 'cfg' argument, containing the ajax request configuration to be modified before the request is sent. The function has to return 'true' to continue with the ajax request; if 'false' or nothing is returned, the ajax request will be aborted.
