ICEfaces ACE Components
Tag animation

The ace:animation tag provides a way to animate standard JSF components. Animations are based on the jQuery effects utilities.

For more information, see the Animation Wiki Documentation.

The TLD documentation of each standard JSF component lists which events are supported, which usually map to standard DOM events. The event name must be entered without the 'on' prefix (e.g. for 'onmousemove' the value of this attribute must be 'mousemove').
