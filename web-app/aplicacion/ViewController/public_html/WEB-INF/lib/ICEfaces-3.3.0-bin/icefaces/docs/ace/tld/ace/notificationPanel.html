ICEfaces ACE Components
Tag notificationPanel

The Notification Panel is a component that displays a multipurpose fixed positioned panel for notification. Any group of JSF content can be placed inside notification panel. If position of panel is out of place, try putting it as last child of body.

For more information, see the NotificationPanel Wiki Documentation.
